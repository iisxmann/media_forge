# Media Forge

Comprehensive media processing tool built with Python. Handles image, video, and audio processing with optional AI-powered features.

## Features

- **Image** — Resize, crop, rotate, flip, filters, watermark, format conversion
- **Video** — Trim, merge, compress, extract frames, add subtitles, format conversion
- **Audio** — Extract, convert, normalize, trim, merge, fade effects
- **AI** — Speech-to-text (Whisper), OCR (Tesseract), auto-captioning
- **REST API** — Built-in API server for all operations
- **CLI** — Full command-line interface

## Quick Start

```bash
pip install -r requirements.txt
python -m mediaforge check
python -m mediaforge --help
```
